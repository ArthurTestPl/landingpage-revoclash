# RevoClash - Landing Page

## Overview
Static, modern landing page for RevoClash - a mobile gaming tournament platform targeting competitive gamers in Brazil. The platform allows users to create and participate in paid tournaments with real money prizes via PIX.

## Project Details
- **Created:** November 2, 2025
- **Type:** Static HTML Website
- **Target Audience:** Competitive mobile gamers in Brazil (18+)
- **Language:** Portuguese (Brazil)

## Design Specifications
- **Color Palette:** 
  - Primary Purple: #6B21A8
  - Dark Purple: #4C1D95
  - Light Purple: #7C3AED
  - Black: #000000
  - Accent Green: #10B981
- **Typography:** 
  - Headers: Orbitron (tech/gaming font)
  - Body: Inter
- **Style:** Modern, gamer-friendly, bold and tech-focused

## Features Implemented
1. **Hero Section** - Eye-catching gradient background with prominent CTAs (Create Tournament / Join Tournament)
2. **How It Works** - 3-step flow explaining the tournament process
3. **Prizes and Formats** - Showcases tournament examples (50 players x R$2) and duel mode (1v1)
4. **Why Mobile** - Explains niche focus and reverse engineering technology
5. **Benefits** - 6 key advantages (PIX payments, security, moderation, etc.)
6. **FAQ** - Interactive accordion with common questions (18+ restriction, payment methods, anti-cheat)
7. **Testimonials** - Fictional user reviews from Brazilian gamers
8. **Footer** - Links, social media, legal information
9. **Age Warning** - Prominent 18+ restriction notice
10. **Responsive Design** - Optimized for mobile and desktop
11. **SEO Optimization** - Portuguese meta title and description

## SEO Meta Tags
- **Title:** "RevoClash - Torneios Mobile Pagos com PIX | Ganhe Dinheiro Real Jogando"
- **Description:** "Crie e participe de torneios mobile pagos com PIX. Duelos 1v1 e campeonatos com prêmios reais. Plataforma segura para gamers competitivos +18. Saque rápido e fácil!"

## Key Highlights
- PIX payment integration for instant deposits and withdrawals
- Tournament example: 50 players x R$2 (prizes: R$50 / R$35 / R$15)
- Duel mode: 1v1 betting where winner takes 2x the stake
- Reverse engineering technology to support API-free mobile games
- Active moderation and anti-cheat systems

```bash

```
